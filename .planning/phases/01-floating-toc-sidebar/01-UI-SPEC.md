## Interaction Contract

### Panel Open

- Trigger: Click `#fab-toc`
- Animation: `transform: translateX(-100%)` → `transform: translateX(0)`, `200ms ease-out`
- Focus: Moves to first `.toc-item` button inside panel
- ARIA: Panel has `role="dialog"` and `aria-label="Table of contents"`

### Panel Close

- Trigger 1: `Escape` key (when panel is open)
- Trigger 2: Click on `#content` area (not panel, not FAB)
- Trigger 3: Click `#fab-toc` while panel is open
- Focus: Returns to `#fab-toc` button

### Heading Navigation

- Trigger: Click any `.toc-item` button
- Action: `scrollIntoView({ behavior: 'smooth', block: 'start' })` on the heading element in the markdown content
- Panel closes after heading click (same behavior as other FAB items)

### Keyboard Navigation

- `Tab` cycles through `.toc-item` buttons within the panel
- `Enter` activates heading click (scroll-to)
- `Escape` closes panel and returns focus to FAB

### Responsive (< 600px)

- Panel width: `100vw` (full width)
- Panel dismiss: tap-outside + Escape (no dedicated close button per D-25)

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| `role="dialog"` | Panel has `role="dialog"` and `aria-label="Table of contents"` (D-20) |
| Heading items as buttons | `<button class="toc-item">` — not `<a>` or `<div>` (D-21) |
| Focus management | Focus to first item on open; return to FAB toggle on close (D-23, D-24) |
| Keyboard navigation | Tab cycles items, Enter activates, Escape closes (D-22) |
| Mobile accessibility | Full-width panel still has `role="dialog"` and closes on Escape (D-25) |
